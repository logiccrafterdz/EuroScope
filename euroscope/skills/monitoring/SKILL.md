# 🏥 Monitoring Skill

## What It Does
Monitors EuroScope system health — checks component status, tracks errors, and provides a dashboard.

## Actions
- `check_health` — Run full system health check
- `track_error` — Log an error event
- `get_status` — Overall system status (healthy/degraded/critical)
- `format_dashboard` — Telegram-formatted health dashboard
