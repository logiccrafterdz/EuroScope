# 📊 Performance Analytics Skill

## What It Does
Tracks all trades and computes advanced performance metrics in real-time.

## Actions
- `compute_metrics` — Calculate all metrics from a list of trades
- `get_snapshot` — Current performance snapshot
- `breakdown` — Performance by strategy / session / day-of-week
- `format_report` — Telegram-formatted performance report
