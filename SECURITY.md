# Security Policy

## Supported Versions

Currently, only the latest version of EuroScope is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| < 5.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within EuroScope, please do not disclose it publicly or open a GitHub issue. Instead, please send an e-mail to **logiccrafterdz@gmail.com**.

All security vulnerabilities will be promptly addressed.

Please include the following information in your report:
* Type of issue (e.g., API key leak, injection, unexpected trade execution)
* Full paths of source file(s) related to the manifestation of the issue
* The location of the affected source code (tag/branch/commit or direct URL)
* Any special configuration required to reproduce the issue
* Step-by-step instructions to reproduce the issue
* Impact of the issue, including how an attacker might exploit the issue

We will try to acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress.
